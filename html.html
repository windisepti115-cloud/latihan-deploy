Apa itu HTML?

HTML (Hypertext Markup Language) adalah bahasa markup standar yang digunakan untuk membuat halaman web dan aplikasi web. HTML bukanlah bahasa pemrograman karena tidak bisa memberikan fungsi yang dinamis. HTML digunakan untuk menyusun bagian paragraf, heading, maupun link pada halaman web.
